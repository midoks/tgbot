### UptimePK

时间网格图风格的监控系统

## 安装脚本
```
curl -fsSL https://raw.githubusercontent.com/midoks/uptimepk/master/scripts/install.sh | bash
```

# dev
```
go env -w GOPROXY=https://goproxy.cn,direct
journalctl -u uptimepk -f
```
